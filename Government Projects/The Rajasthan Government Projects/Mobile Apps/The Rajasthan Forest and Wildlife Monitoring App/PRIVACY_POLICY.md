# Privacy Policy

## 1. Data Collection

We collect:

- **Biometric Data:** Fingerprint/Face ID for authentication (stored locally).
- **Location Data:** GPS coordinates for patrol logging (Gasti Path).
- **Media:** Audio and Images for evidence.

## 2. Data Security

- All sensitive data is encrypted using **AES-256-GCM** ('Vajra-Kavach') before storage.
- No keys are stored on the server; they are derived from authorized user credentials.

## 3. Data Sharing

- Data is shared strictly with the **State Data Center (SDC)** and **MoEFCC** repositories via secure APIs.
- We do not sell or share data with third-party advertisers.
