# Contributing to Raj Kissan X (OpenForge Copy)

Thank you for your interest in contributing to the **Raj Kissan Organic Agriculture System**.

## Code of Conduct

This project is governed by the [Code of Conduct](CODE_OF_CONDUCT.md).

## How to Contribute

### 1. Reporting Bugs

- Open a new Issue on OpenForge/GitHub.
- **Do not** include any Real Farmer Data (Kissan ID/Land Records). Use dummy data.

### 2. Suggesting Enhancements

- Open an Issue with the tag `enhancement`.
- Explain benefit for **Krishi Sakhis** or **Farmers**.

### 3. Pull Requests (PR)

- Fork the repository.
- `git checkout -b feature/AmazingFeature`.
- `git push origin feature/AmazingFeature`.
- Open a Pull Request.

## Coding Standards (Vastu-Digital)

- **JavaScript**: ES6+ features.
- **Encryption**: NEVER bypass the `EncryptionService`. Land Records must be encrypted.
- **Language**: Variable names in English, UI strings supported in Hindi/Rajasthani.

## Review Process

1.  **Lead Architect** (ChitraHarsha VPK)
2.  **Agri-Science Lead** (KVK Expert)
3.  **Project Manager** (Agriculture Dept)
